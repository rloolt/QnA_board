QnA 게시판 프로젝트 v3

본 프로젝트는 Flask와 MySQL을 활용한 웹 게시판 서비스입니다. 
이전 프로젝트의 문제점들을 파악하고, 도커 환경에서 보안성과 게시판 기능을 추가하였습니다.

<주요 수정 내용>
1. 보안 및 설정 관리
- 환경 변수(.env) 도입: DB 비밀번호, Flask Secret Key 등 민감 정보를 코드에서 분리하여 보안성 향상.
- 하드코딩 제거: 모든 설정값을 환경 변수 파일로 관리하여 코드 수정 없이 환경 변경 가능.

2. 게시판 기능 강화
- 프로필 시스템: 사용자 프로필 조회 및 수정 기능 추가.
- 이미지 업로드: 프로필 사진 업로드 및 게시글 파일 첨부 기능 구현.
- 비밀글 설정: 게시글 작성 시 비밀번호를 설정하여 작성자와 비밀번호를 아는 사용자만 열람 가능.
- 파일 첨부 및 다운로드: 게시글에 파일 첨부 기능을 추가하고, 서버 내 안전한 저장 및 다운로드 로직 구현.

3. 아이디/비밀번호 찾기
- 유효성 검사를 통한 계정 정보 복구 로직 구현.
